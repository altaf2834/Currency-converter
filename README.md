# Currency Converter

A responsive Currency Converter web application built using HTML, CSS, and JavaScript. This project allows users to convert amounts between different currencies using real-time exchange rates and provides a clean, user-friendly interface.

## Features

- Real-time currency conversion
- Support for multiple currencies
- One-click currency swap functionality
- Responsive and modern user interface
- Easy-to-use design
- Fast and lightweight application

## How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/altaf2834/Currency-converter.git
```

2. Navigate to the project folder

```bash
cd Currency-converter
```

3. Open `index.html` in your browser
